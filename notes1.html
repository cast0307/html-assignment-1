5 HTML Tags

For the following content you will find 5 learnt html tags and their respective description:

Paragraph tags
These are tags to create paragraphs.
Images tags
These are tags to reference an image that is saved on our files.
Anchors tags
These are tags to link everything in our website.
Comment tags
These are tags to put comments in our coding
Navigation tags
These are tags to create navigation bars between our pages.
